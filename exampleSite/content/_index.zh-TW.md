# Hugo Theme Deca

歡迎使用 Deca - Hugo 生態系中首個支援 SPA 功能的現代化文件主題。

Deca 的設計目標是在 Hugo 的高效能基礎上，提供接近 Docusaurus 和 Vitepress 的使用體驗，讓你能建立專業且流暢的文件網站。
